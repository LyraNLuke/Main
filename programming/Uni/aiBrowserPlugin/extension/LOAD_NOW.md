# ✅ Load Extension Into Chrome - Checklist

## Right Now (2 minutes)

- [ ] Open Chrome (or Edge, Brave)
- [ ] Go to: `chrome://extensions/`
- [ ] Turn ON "Developer mode" (top right)
- [ ] Click "Load unpacked" button
- [ ] Navigate to and select: `extension/dist/` folder
- [ ] Click "Select Folder"

## Verify It Works

- [ ] Extension appears in extensions list
- [ ] Go to any website
- [ ] See purple button in bottom-right corner
- [ ] Click it - popup opens
- [ ] Type a task: "Learn Python"
- [ ] Click "Break Down Task"
- [ ] See subtasks appear
- [ ] Click one to expand
- [ ] Click "Start Task"
- [ ] See 5-second countdown
- [ ] Stopwatch starts automatically
- [ ] Click "Mark Done"
- [ ] Task marked as complete ✓

## Done! 🎉

You now have a working AI task breakdown extension with no setup!

Drag the button anywhere, it works on every website.
